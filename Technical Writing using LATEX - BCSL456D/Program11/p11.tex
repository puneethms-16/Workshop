\section*{Algorithm Example}  
Below is an example of an algorithm written in LaTeX using the algorithm2e.  
\begin{algorithm}[H]  
\SetAlgoLined   
\KwData{Input data for the algorithm}  
\KwResult{Output of the algorithm}  
\KwIn{A list of integers, \( A = [a_1, a_2, \dots, a_n] \)}  
\KwOut{The maximum value in the list}  
\caption{Find the Maximum Value in a List}  
\Begin{  
\SetKwFunction{FMain}{FindMax}  
\SetKwProg{Fn}{Function}{:}{}  
\Fn{\FMain{$A$}}{  
$max \gets A[0]$\; 
\For{$i \gets 1$ \textbf{to} length($A$) - 1}{  
\If{$A[i] > max$}{  
$max \gets A[i]$\;  
}  
}  
\KwRet{$max$}\;  
}  
\KwRet{Call \FMain{$A$}}\;  
}  
\end{algorithm}
